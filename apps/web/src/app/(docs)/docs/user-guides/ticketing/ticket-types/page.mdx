# Ticket Types

Configure and manage the types of tickets available for your attraction.

![Ticket Types](/docs/screenshots/ticket-types.png)

## Overview

Ticket types define the different admission options guests can purchase for your attraction. Each type can have its own pricing, availability, and capacity settings.

## Creating a Ticket Type

1. Navigate to **Ticketing** in the sidebar
2. Click **+ New Ticket Type**
3. Fill in the details:
   - **Name**: Display name (e.g., "General Admission")
   - **Description**: What's included
   - **Price**: Ticket price in dollars
   - **Capacity**: Max tickets per time slot (optional)
4. Configure additional settings
5. Click **Save**

## Ticket Type Settings

### Basic Information

| Field | Description |
|-------|-------------|
| Name | Ticket name shown to customers |
| Description | Details about the ticket |
| Price | Base price before fees |
| Active | Whether tickets can be purchased |

### Availability Settings

| Field | Description |
|-------|-------------|
| Valid Days | Which days this ticket is available |
| Valid Times | Time slots when valid |
| Start/End Date | Seasonal availability |
| Max Per Order | Limit tickets per transaction |

### Capacity Management

- **Unlimited**: No capacity restrictions
- **Per Time Slot**: Limited availability per entry time
- **Daily Cap**: Maximum sold per day

## Common Ticket Types

### General Admission
Standard entry ticket at regular price.

### VIP / Fast Pass
Premium tickets with benefits like:
- Skip the line access
- Priority entry
- Exclusive experiences

### Group Tickets
Discounted tickets for groups:
- Minimum group size required
- Bulk pricing available

### Special Events
Limited tickets for specific events:
- Date-restricted
- May include add-ons

## Managing Ticket Types

### Edit a Ticket Type

1. Click on the ticket type
2. Update the settings
3. Save changes

Changes apply to future sales only.

### Deactivate a Ticket Type

1. Open the ticket type
2. Toggle **Active** to off
3. Ticket will no longer be available for purchase

Existing tickets remain valid.

### Delete a Ticket Type

Only delete if no tickets have been sold:

1. Open the ticket type
2. Click **Delete**
3. Confirm deletion

## Pricing Strategies

### Dynamic Pricing

Adjust prices based on demand:
- Weekend premium
- Holiday pricing
- Early bird discounts

### Tiered Pricing

Different prices for different groups:
- Adult vs. Child
- Senior discounts
- Student pricing
