# Ticket Types

Configure and manage the types of tickets available for your attractions.

## Overview

Ticket types define the different admission options guests can purchase for your attractions. Each type can have its own pricing, capacity settings, category assignment, and included amenities.

## Accessing Ticket Types

1. Navigate to **Ticketing** in the sidebar
2. Click **Ticket Types**

## Ticket Types List

The list displays all configured ticket types:

| Column | Description |
|--------|-------------|
| Name | Ticket name and description |
| Category | Assigned category (if any) |
| Attraction | Which attraction this ticket is for |
| Price | Current price (with compare price if set) |
| Sold | Number sold / capacity (if set) |
| Status | Active or Inactive |
| Actions | Edit, activate/deactivate, delete |

## Creating a Ticket Type

1. Click **Add Ticket Type** button
2. Fill in the ticket details
3. Click **Create**

### Ticket Type Fields

| Field | Required | Description |
|-------|----------|-------------|
| Name | Yes | Display name (e.g., "General Admission") |
| Description | No | What's included or details about the ticket |
| Attraction | Yes | Which attraction this ticket is valid for |
| Category | No | Organize tickets by category |
| Price | Yes | Base ticket price in dollars |
| Compare Price | No | Original price for strikethrough display |
| Min Per Order | No | Minimum tickets per transaction |
| Max Per Order | No | Maximum tickets per transaction |
| Capacity | No | Total available tickets |
| Includes | No | List of included items (one per line) |

### Pricing Fields

**Price**: The current selling price for the ticket.

**Compare Price**: An optional higher price that will be shown with a strikethrough, creating a perception of value. Use for sales or promotions.

### Order Quantity Limits

**Min Per Order**: The minimum number of this ticket type that must be purchased in a single transaction. Useful for group tickets.

**Max Per Order**: The maximum number of this ticket type that can be purchased in a single transaction. Helps manage capacity.

### Includes List

Enter one item per line to describe what's included with the ticket:
- Access to all attractions
- Photo package
- Souvenir item
- Priority entry

This list displays on the storefront to help guests understand the value.

## Editing a Ticket Type

1. Find the ticket type in the list
2. Click the menu button (...)
3. Select **Edit**
4. Modify the desired fields
5. Click **Save Changes**

Changes apply to future sales only. Existing tickets are not affected.

## Activating/Deactivating Ticket Types

### To Deactivate

1. Find the ticket type
2. Click the menu button (...)
3. Select **Deactivate**

Deactivated ticket types:
- Cannot be purchased
- Do not appear on storefront
- Existing tickets remain valid

### To Activate

1. Find the deactivated ticket type
2. Click the menu button (...)
3. Select **Activate**

The ticket type will immediately be available for purchase.

## Deleting a Ticket Type

1. Find the ticket type
2. Click the menu button (...)
3. Select **Delete**
4. Confirm the deletion

**Note**: You cannot delete a ticket type that has been purchased. Deactivate it instead.

## Categories

Categories help organize your ticket types:
- Group similar tickets together
- Display category badges in the list
- Can be filtered on storefront

Categories are managed separately and assigned to ticket types.

## Capacity Management

### Unlimited Capacity

Leave the capacity field empty for unlimited tickets.

### Limited Capacity

Set a specific number to limit total sales:
- Sold count tracks purchases
- Shows "X / Y" format (sold / capacity)
- Cannot sell more than capacity

When capacity is reached:
- Ticket type shows as sold out
- No additional purchases allowed
- Consider creating a waitlist

## Pricing Strategies

### Standard Pricing

Set a single price for all guests.

### Tiered Pricing

Create multiple ticket types at different price points:

| Ticket Type | Price | Description |
|-------------|-------|-------------|
| Child (3-12) | $15.00 | For children ages 3-12 |
| Adult | $25.00 | Standard adult admission |
| Senior (65+) | $20.00 | Senior discount |

### Sale Pricing

Use compare price for promotional pricing:

| Price | Compare Price | Display |
|-------|---------------|---------|
| $19.99 | $29.99 | ~~$29.99~~ $19.99 |

### VIP/Premium Tiers

Create premium experiences with higher pricing:

| Ticket Type | Price | Includes |
|-------------|-------|----------|
| General Admission | $25.00 | Standard entry |
| VIP | $50.00 | Priority entry, exclusive areas, photo |
| Ultimate | $100.00 | All VIP benefits plus meet & greet |

## Ticket Type Examples

### General Admission

```
Name: General Admission
Description: Standard entry to all attraction areas
Attraction: Nightmare Manor
Price: $29.99
Max Per Order: 10
Includes:
  - Full attraction access
  - 2-3 hour experience
```

### Fast Pass

```
Name: Fast Pass
Description: Skip the line with priority entry
Attraction: Nightmare Manor
Price: $49.99
Compare Price: $59.99
Capacity: 100
Max Per Order: 4
Includes:
  - Priority entry (no wait)
  - Full attraction access
  - Exclusive photo opportunity
```

### Group Package

```
Name: Group Package (10+)
Description: Discounted rate for groups of 10 or more
Attraction: Nightmare Manor
Price: $22.99
Min Per Order: 10
Max Per Order: 50
Includes:
  - Full attraction access
  - Group coordinator
  - Reserved entry time
```

## Best Practices

### Naming

- Use clear, descriptive names
- Include age ranges if applicable
- Mention key benefits (VIP, Fast Pass)

### Descriptions

- Keep descriptions concise
- Highlight unique value
- List key inclusions

### Pricing

- Research competitor pricing
- Consider perceived value
- Use compare prices strategically

### Capacity

- Set realistic limits
- Monitor sales velocity
- Adjust capacity as needed

## Troubleshooting

### Cannot Delete Ticket Type

- Ticket type has been purchased
- Deactivate instead of deleting
- Historical data is preserved

### Price Not Updating

- Changes apply to new orders only
- Refresh the page
- Check for validation errors

### Ticket Not Appearing on Storefront

- Verify ticket is active
- Check attraction is published
- Confirm category is visible
