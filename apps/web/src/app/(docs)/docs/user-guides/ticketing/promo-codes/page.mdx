# Promo Codes

Create and manage discount codes for your ticket sales.

## Overview

Promo codes allow you to offer discounts to customers. Use them for marketing campaigns, special promotions, partner deals, or to reward loyal guests.

## Accessing Promo Codes

1. Navigate to **Ticketing** in the sidebar
2. Click **Promo Codes**

## Promo Codes List

The list displays all configured promo codes:

| Column | Description |
|--------|-------------|
| Code | The promo code (with copy button) |
| Discount | Discount amount or percentage |
| Uses | Times used / max uses |
| Valid Period | Date range or "Always valid" |
| Status | Active, Inactive, Expired, Scheduled, Exhausted |
| Actions | Edit, activate/deactivate, delete |

## Code Statuses

| Status | Badge | Description |
|--------|-------|-------------|
| Active | Green | Code is currently valid and usable |
| Inactive | Gray | Manually deactivated |
| Expired | Red | Past the valid until date |
| Scheduled | Outline | Not yet reached valid from date |
| Exhausted | Red | Reached maximum uses |

## Creating a Promo Code

1. Click **Create Promo Code** button
2. Fill in the code details
3. Click **Create**

### Promo Code Fields

| Field | Required | Description |
|-------|----------|-------------|
| Code | Yes | The text customers enter |
| Description | No | Internal notes about the promotion |
| Discount Type | Yes | Percentage or Fixed Amount |
| Discount Value | Yes | Percentage (1-100) or dollar amount |
| Min Order | No | Minimum order amount to apply |
| Max Discount | No | Maximum discount cap (for percentage) |
| Max Total Uses | No | Total times code can be used |
| Max Per Customer | No | Limit per unique customer |
| Valid From | No | Start date for the promotion |
| Valid Until | No | Expiration date |

### Code Field

The promotional code text that customers will enter:
- Automatically converted to uppercase
- Should be memorable and easy to type
- Avoid confusing characters (0/O, 1/l)

**Generate Button**: Click **Generate** to create a random 8-character code.

### Discount Types

**Percentage**: Takes a percentage off the order total.
- Value: 1-100
- Example: 20% off

**Fixed Amount**: Takes a fixed dollar amount off.
- Value: Dollar amount
- Example: $10 off

### Usage Limits

**Max Total Uses**: The total number of times this code can be redeemed across all customers.

**Max Per Customer**: The number of times a single customer can use this code.

### Minimum Order Amount

Require a minimum order value before the code applies:
- Encourages larger purchases
- Prevents abuse on small orders
- Shows "Min: $X.XX" in the list

### Maximum Discount

For percentage codes, cap the maximum discount amount:
- Protects against large order discounts
- Useful for high-percentage codes

## Editing a Promo Code

1. Find the promo code in the list
2. Click the menu button (...)
3. Select **Edit**
4. Modify the desired fields
5. Click **Save Changes**

**Note**: The code text cannot be changed after creation.

## Copying a Code

Click the copy icon next to any code to copy it to your clipboard.

A toast notification confirms: `Copied "CODE" to clipboard`

## Activating/Deactivating Codes

### To Deactivate

1. Find the promo code
2. Click the menu button (...)
3. Select **Deactivate**

Deactivated codes will not work at checkout.

### To Activate

1. Find the deactivated code
2. Click the menu button (...)
3. Select **Activate**

The code will immediately work again (if within valid dates and usage limits).

## Deleting a Promo Code

1. Find the promo code
2. Click the menu button (...)
3. Select **Delete**
4. Confirm the deletion

**Note**: Cannot delete codes that have been used. Deactivate instead.

## Promo Code Examples

### Percentage Discount

```
Code: SCARY20
Discount Type: Percentage
Discount Value: 20%
Valid Until: October 31, 2024
```
Takes 20% off the order total until Halloween.

### Fixed Amount

```
Code: SAVE10
Discount Type: Fixed Amount
Discount Value: $10.00
Min Order: $30.00
```
Takes $10 off orders of $30 or more.

### Limited Campaign

```
Code: PARTNER2024
Discount Type: Percentage
Discount Value: 15%
Max Total Uses: 500
Max Per Customer: 1
Valid From: October 1, 2024
Valid Until: October 31, 2024
```
Partner promotion with limited redemptions.

### VIP Code

```
Code: VIP50OFF
Discount Type: Fixed Amount
Discount Value: $50.00
Max Per Customer: 1
Min Order: $100.00
```
VIP discount for high-value orders.

### Flash Sale

```
Code: FLASH25
Discount Type: Percentage
Discount Value: 25%
Max Discount: $20.00
Valid From: [today]
Valid Until: [tomorrow]
```
24-hour flash sale with capped discount.

## Tracking Performance

The uses column shows redemption stats:
- Current uses vs. maximum
- Red text when limit reached
- "Exhausted" status when maxed out

Track which codes perform best for future campaigns.

## Best Practices

### Code Design

- Use memorable codes: HALLOWEEN24, SCARYNIGHT
- Include campaign name for tracking
- Avoid confusing characters

### Time Limits

- Set expiration dates to create urgency
- Use "Valid From" for scheduled promotions
- Consider time zones

### Usage Limits

- Limit total uses for exclusive offers
- Limit per customer to prevent abuse
- Monitor usage velocity

### Testing

- Test codes before launching campaigns
- Verify discount calculates correctly
- Check with various order amounts

## Troubleshooting

### Code Not Working

- Check code is active
- Verify within valid date range
- Confirm usage limits not reached
- Check minimum order requirements
- Ensure correct code entered (case-insensitive)

### Cannot Delete Code

- Code has been used
- Deactivate instead
- Historical data is preserved

### Wrong Discount Applied

- Check discount type (percentage vs. fixed)
- Verify discount value
- Check for maximum discount cap
