# Promo Codes

Create and manage discount codes for your ticket sales.

![Promo Codes](/docs/screenshots/promo-codes.png)

## Overview

Promo codes allow you to offer discounts to customers. Use them for marketing campaigns, special promotions, or partner deals.

## Creating a Promo Code

1. Navigate to **Ticketing** â†’ **Promo Codes**
2. Click **+ New Promo Code**
3. Configure the code:
   - **Code**: The text customers enter (e.g., "SAVE20")
   - **Discount Type**: Percentage or fixed amount
   - **Discount Value**: How much off
4. Set restrictions and limits
5. Click **Save**

## Promo Code Settings

### Discount Configuration

| Setting | Description |
|---------|-------------|
| Code | The promotional code text |
| Type | Percentage or fixed amount |
| Value | Discount amount or percentage |
| Min Purchase | Minimum order value required |

### Usage Limits

| Setting | Description |
|---------|-------------|
| Total Uses | Max times code can be used |
| Uses Per Customer | Limit per customer |
| Start Date | When code becomes active |
| End Date | When code expires |

### Restrictions

- **Specific Ticket Types**: Only apply to certain tickets
- **Specific Days**: Valid only on certain days
- **First-Time Only**: New customers only
- **Combinable**: Can stack with other codes

## Promo Code Examples

### Percentage Discount
```
Code: SCARY20
Type: Percentage
Value: 20%
```
Takes 20% off the order total.

### Fixed Amount
```
Code: 10OFF
Type: Fixed
Value: $10
```
Takes $10 off the order.

### Partner Code
```
Code: PARTNER2024
Type: Percentage
Value: 15%
Uses: 500
```
Limited-use partner discount.

## Managing Promo Codes

### View Usage Statistics

Each code shows:
- Times used
- Total discount given
- Revenue from orders using code

### Edit a Promo Code

1. Click on the promo code
2. Update settings
3. Save changes

### Deactivate a Code

1. Open the promo code
2. Toggle **Active** off
3. Code will no longer work

### Delete a Code

1. Open the promo code
2. Click **Delete**
3. Confirm deletion

Historical usage data is preserved.

## Best Practices

1. **Use memorable codes**: HALLOWEEN2024, SCARYNIGHT
2. **Set expiration dates**: Create urgency
3. **Track performance**: Monitor which codes convert best
4. **Limit usage**: Prevent abuse with caps
5. **Test before launch**: Verify code works correctly
