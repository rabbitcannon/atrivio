# Orders

View and manage ticket orders for your attraction.

![Orders List](/docs/screenshots/orders-list.png)

## Overview

The orders page provides a complete view of all ticket purchases, allowing you to track sales, process refunds, and manage customer orders.

## Viewing Orders

### Order List

The main orders view shows:
- Order number
- Customer name
- Order date
- Total amount
- Status (completed, refunded, pending)

### Filtering Orders

Filter by:
- **Date range**: View specific time periods
- **Status**: Completed, refunded, pending
- **Ticket type**: Filter by product
- **Payment method**: Card, cash, etc.

### Searching Orders

Search by:
- Order number
- Customer email
- Customer name

## Order Details

Click an order to view:

### Order Information
- Order number and date
- Payment status
- Payment method
- Transaction ID

### Customer Information
- Name and email
- Phone number (if provided)
- Order history link

### Ticket Details
- Ticket type(s) purchased
- Quantity of each
- Unit price and subtotal
- Event date/time

### Financial Summary
- Subtotal
- Discounts applied
- Fees
- Total charged

## Processing Refunds

### Full Refund

1. Open the order
2. Click **Refund Order**
3. Select **Full Refund**
4. Add refund reason
5. Confirm refund

### Partial Refund

1. Open the order
2. Click **Refund Order**
3. Select tickets to refund
4. Enter refund amount
5. Add reason
6. Confirm

### Refund Notes

- Refunds process to original payment method
- May take 5-10 business days to appear
- Refunded tickets are invalidated

## Resending Tickets

If a customer needs their tickets again:

1. Open the order
2. Click **Resend Tickets**
3. Confirm the email address
4. Click **Send**

## Order Status

| Status | Description |
|--------|-------------|
| Completed | Payment successful, tickets delivered |
| Pending | Awaiting payment confirmation |
| Refunded | Full refund processed |
| Partially Refunded | Partial refund processed |
| Cancelled | Order cancelled before completion |

## Exporting Orders

Export order data for reporting:

1. Set your filters
2. Click **Export**
3. Choose format (CSV or Excel)
4. Download the file

Export includes:
- Order details
- Customer information
- Payment data
- Ticket information

## Analytics

View order metrics:
- Total revenue
- Average order value
- Orders by day/week/month
- Top-selling ticket types
- Promo code performance
