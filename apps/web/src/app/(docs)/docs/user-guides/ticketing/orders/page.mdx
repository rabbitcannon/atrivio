# Orders

View and manage ticket orders for your attractions.

## Overview

The orders page provides a complete view of all ticket purchases, allowing you to track sales, view order details, and process completions, cancellations, and refunds.

## Accessing Orders

1. Navigate to **Ticketing** in the sidebar
2. Click **Orders**

## Orders List

The list displays all orders with key information:

| Column | Description |
|--------|-------------|
| Order | Order number (with promo badge if used) |
| Customer | Email and name |
| Attraction | Which attraction the order is for |
| Total | Order total (with discount if applied) |
| Status | Current order status |
| Date | When the order was placed |
| Actions | View details, complete, cancel, refund |

## Order Statuses

| Status | Color | Description |
|--------|-------|-------------|
| Pending | Gray | Awaiting payment or manual completion |
| Processing | Outline | Payment being processed |
| Completed | Green | Payment successful, tickets generated |
| Canceled | Red | Order canceled before completion |
| Refunded | Red | Full refund processed |

## Searching Orders

### By Email

Enter a customer email in the search box and click **Search** (or press Enter).

### By Status

Use the status dropdown to filter:
- All statuses
- Pending
- Processing
- Completed
- Canceled
- Refunded

## Viewing Order Details

1. Find the order in the list
2. Click the menu button (...)
3. Select **View Details**

### Order Details Dialog

The details dialog shows comprehensive order information:

#### Customer Information
- Email address
- Name (if provided)
- Phone number (if provided)

#### Status
Current order status with color badge.

#### Items Table
| Column | Description |
|--------|-------------|
| Item | Ticket type name |
| Time Slot | Date and time (if applicable) |
| Qty | Quantity purchased |
| Price | Unit price |
| Total | Line item total |

#### Financial Summary
- Subtotal
- Discount (with promo code shown)
- Total charged

#### Tickets (for completed orders)
List of generated tickets showing:
- Barcode
- Ticket type
- Status (valid, used, voided)

## Order Actions

### Complete Order

For pending orders:

1. Click the menu button (...)
2. Select **Complete Order**
3. Order is marked complete and tickets are generated

Use when manually confirming payment or for comp tickets.

### Cancel Order

For pending or processing orders:

1. Click the menu button (...)
2. Select **Cancel**
3. Confirm the cancellation

Canceled orders:
- Cannot be completed later
- No tickets are generated
- No charge is processed

### Refund Order

For completed orders:

1. Click the menu button (...)
2. Select **Refund**
3. Confirm: "Refund order? This will void all tickets."

Refunded orders:
- Funds returned to customer
- All tickets are voided
- Cannot be reversed

## Promo Code Display

Orders using promo codes show:
- Promo badge next to order number
- Discount amount in green
- Promo code name in details

## Pagination

For large order volumes:
- Navigate between pages using Previous/Next buttons
- Page count shown: "Page X of Y"
- 20 orders displayed per page

## Ticket Information

Completed orders show generated tickets:

| Field | Description |
|-------|-------------|
| Barcode | Unique ticket identifier |
| Type | Ticket type name |
| Status | Valid, Used, or Voided |

Ticket statuses:
- **Valid**: Ready for use
- **Used**: Already scanned at check-in
- **Voided**: Canceled or refunded

## Best Practices

### Order Monitoring

- Check pending orders regularly
- Complete legitimate orders promptly
- Cancel suspicious orders quickly

### Refund Processing

- Process refunds within business policy
- Document refund reasons
- Verify customer identity if needed

### Customer Communication

- Respond to order inquiries promptly
- Provide order numbers in communications
- Direct customers to support for issues

## Common Workflows

### Walk-Up Sale

1. Create order at box office
2. Collect payment manually
3. Click Complete Order
4. Print or email tickets

### Disputed Charge

1. Find the order
2. Review order details
3. Verify with customer
4. Process refund if appropriate

### Event Cancellation

1. Filter by event date
2. Review affected orders
3. Process refunds as needed
4. Notify customers

## Troubleshooting

### Order Not Found

- Check email spelling
- Try different search terms
- Clear filters and search again

### Cannot Complete Order

- Check payment status
- Verify order is pending
- Contact payment support

### Cannot Refund Order

- Check order is completed
- Verify Stripe account status
- Review order for prior refunds

### Tickets Not Generating

- Verify order was completed
- Check ticket type is active
- Review system logs
