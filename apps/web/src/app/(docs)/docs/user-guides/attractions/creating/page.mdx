# Creating Attractions

Add new attractions to your organization to manage venues, sell tickets, and schedule staff.

## Before You Start

Ensure you have:

- **Owner or Admin role** in your organization
- A clear name and type for your attraction
- Basic details like capacity and duration (optional but recommended)

## Create an Attraction

1. Navigate to **Attractions** in the sidebar
2. Click the **Add Attraction** button
3. Fill in the required fields:
   - **Name** - Display name (e.g., "Nightmare Manor")
   - **URL Slug** - Auto-generated from name (e.g., "nightmare-manor")
   - **Type** - Select from available attraction types
4. Add optional details (see below)
5. Click **Create Attraction**

## Required Fields

### Name

The display name shown throughout the platform:

- Guest-facing storefront
- Staff schedules
- Reports and analytics
- Ticket listings

**Best practices:**
- Keep it memorable and clear
- Avoid special characters
- Maximum 100 characters

### URL Slug

The URL-friendly version of your attraction name:

- Auto-generated from the name
- Used in storefront URLs: `yourstore.atrivio.app/nightmare-manor`
- Cannot be changed after creation
- Only lowercase letters, numbers, and hyphens

### Type

Select the category that best describes your attraction:

| Type | Best For |
|------|----------|
| Haunted House | Indoor walkthrough experiences |
| Haunted Trail | Outdoor walking paths |
| Escape Room | Puzzle-based timed rooms |
| Hayride | Vehicle-based attractions |
| Corn Maze | Outdoor maze experiences |
| Other | Custom or unique attractions |

## Optional Fields

### Capacity

Maximum guests allowed simultaneously:

- Used for check-in management
- Displayed on storefront
- Helps with scheduling decisions
- Leave blank if no limit

### Minimum Age

Age restriction for guests:

- Displayed on storefront as a warning
- Used in ticket purchase flow
- Set to 0 or leave blank for all ages

### Intensity Level (1-5)

Scare/intensity rating:

| Level | Description |
|-------|-------------|
| 1 | Family-friendly, minimal scares |
| 2 | Light scares, younger teens OK |
| 3 | Moderate intensity, teens+ |
| 4 | High intensity, adults recommended |
| 5 | Extreme, not for the faint of heart |

### Duration (Minutes)

Typical guest experience time:

- Displayed on storefront
- Helps guests plan their visit
- Used for throughput calculations

### Description

A brief description of your attraction:

- Shown on the attraction detail page
- Can be expanded in storefront settings
- Supports plain text only

## After Creation

Once your attraction is created:

1. **Set up Zones** - Define areas for staff assignments
2. **Configure Seasons** - Set operating dates and hours
3. **Build Storefront** - Create your public-facing site
4. **Add Ticket Types** - Configure pricing and packages
5. **Assign Staff** - Schedule actors and crew

## Example: Creating a Haunted House

```
Name: Victorian Manor of Horrors
Slug: victorian-manor (auto-generated)
Type: Haunted House
Capacity: 25
Min Age: 12
Intensity: 4
Duration: 20 minutes
Description: A terrifying journey through a 19th century mansion...
```

## Tier Limits

Your subscription tier limits how many attractions you can create:

| Tier | Limit | Action When Reached |
|------|-------|---------------------|
| Free | 1 | Upgrade to Pro |
| Pro | 5 | Upgrade to Enterprise |
| Enterprise | Unlimited | N/A |
