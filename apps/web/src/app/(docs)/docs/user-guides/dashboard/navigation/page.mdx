# Dashboard Navigation

Learn how to navigate the Atrivio dashboard efficiently.

## Sidebar Navigation

The sidebar is your main navigation tool. It's organized into logical sections:

### Core Features
- **Dashboard** - Your home base with metrics
- **Attractions** - Manage experiences
- **Staff** - Team management
- **Schedule** - Shift planning

### Operations
- **Ticketing** - Ticket types, orders, promo codes
- **Check-In** - Guest entry management
- **Queue** - Virtual queue (Enterprise)
- **Inventory** - Props and costumes (Pro)

### Settings
- **Organization** - Business settings
- **Payments** - Stripe configuration
- **Storefront** - Public website
- **Notifications** - Alert preferences

## Keyboard Shortcuts

Speed up your workflow with keyboard shortcuts:

| Shortcut | Action |
|----------|--------|
| `G` then `D` | Go to Dashboard |
| `G` then `S` | Go to Staff |
| `G` then `T` | Go to Ticketing |
| `G` then `C` | Go to Check-In |
| `/` | Open search |
| `?` | Show all shortcuts |

## Search

Press `/` or click the search bar to search for:

- Staff members
- Ticket orders
- Attractions
- Settings

Results appear instantly as you type.

## Breadcrumbs

At the top of each page, breadcrumbs show your current location:

```
Dashboard > Staff > John Smith > Edit Profile
```

Click any breadcrumb to navigate back.

## Notifications

The bell icon shows recent notifications:

- Shift swap requests
- New team member invitations
- System alerts

Click a notification to go directly to the relevant page.

## User Menu

Click your avatar in the top-right to access:

- **Profile** - Edit your personal info
- **Settings** - Your preferences
- **Help** - Documentation and support
- **Sign Out** - Log out of your account
