# Dashboard Overview

The dashboard is your command center for managing your attraction. Here's what you'll find.

## Main Dashboard

When you log in, you'll see the main dashboard with key metrics:

![Dashboard overview](/docs/screenshots/dashboard-overview.png)

### Quick Stats
- **Total Staff** - Number of team members
- **Active Shifts** - Staff currently working
- **Today's Check-ins** - Guests checked in today
- **Revenue** - Today's ticket sales

### Recent Activity
A feed of recent actions:
- Staff clock-ins and clock-outs
- Ticket purchases
- Check-ins
- Schedule changes

## Navigation

The sidebar provides quick access to all features:

| Section | Description |
|---------|-------------|
| **Dashboard** | Overview and quick stats |
| **Attractions** | Manage your attractions |
| **Staff** | Team members and profiles |
| **Schedule** | Shift management |
| **Ticketing** | Ticket types and orders |
| **Check-In** | Guest check-in stations |
| **Settings** | Organization settings |

## Quick Actions

From the dashboard, you can quickly:

- **Clock In/Out** - For staff members
- **Start Check-In** - Open a check-in station
- **View Today's Schedule** - See who's working
- **Create Ticket Order** - Process walk-up sales

## Switching Organizations

If you belong to multiple organizations:

1. Click your organization name in the header
2. Select a different organization from the dropdown
3. The dashboard will update to show that organization's data

## Customizing Your View

Your dashboard remembers your preferences:

- **Date range** - Filter stats by time period
- **Attraction filter** - Focus on specific attractions
- **Dark mode** - Toggle in settings
