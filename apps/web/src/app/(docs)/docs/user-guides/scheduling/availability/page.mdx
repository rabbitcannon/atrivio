# Staff Availability

View and manage team availability and time-off requests.

## Overview

The Staff Availability view shows when each team member is available to work. This helps schedulers create efficient schedules that respect staff preferences and time-off requests.

## Accessing Availability

1. Navigate to **Schedule** in the sidebar
2. Click **Staff Availability**

## Availability Matrix

The availability matrix displays a grid view:

| Component | Description |
|-----------|-------------|
| Rows | Each staff member |
| Columns | Days of the week (Sun-Sat) |
| Cells | Availability status for that day |

## Availability Types

| Type | Icon | Color | Description |
|------|------|-------|-------------|
| Available | Checkmark | Green | Can work this day |
| Preferred | Checkmark | Dark Green | Prefers to work this day |
| Unavailable | X | Red | Cannot work this day |
| Time Off (Approved) | Minus | Yellow | Approved time off |
| Time Off (Pending) | Clock | Orange | Requested, awaiting approval |
| Not Set | Dash | Gray | No availability specified |

## Reading the Matrix

### Staff Row

Each row shows:
- Staff avatar and name
- Role badge
- Availability for each day

### Day Columns

Each column represents a day of the week:
- Sunday through Saturday
- Shows icon indicating availability type
- May show time range if partial availability

### Time Ranges

When staff has partial availability:
- Start and end times are shown
- Example: "09:00-17:00" means available 9 AM to 5 PM
- Full day availability shows no time range

## Staff Setting Availability

Staff members can set their availability:

1. Navigate to their schedule view
2. Select a day
3. Choose availability type
4. Set time range (optional)
5. Save changes

### Recurring Availability

Set weekly patterns:
- Same availability each week
- Define typical working days
- Update as needed

### One-Time Changes

Request specific date changes:
- Vacation days
- Appointments
- Special circumstances

## Time-Off Requests

### Requesting Time Off

Staff members can:
1. Select the date(s)
2. Choose "Request Time Off"
3. Provide reason (optional)
4. Submit request

### Request Status

| Status | Description |
|--------|-------------|
| Pending | Awaiting manager review |
| Approved | Time off granted |
| Rejected | Request denied |

### Managing Requests

Managers can:
1. View pending requests
2. Check schedule impact
3. Approve or reject
4. Add response notes

## Using Availability for Scheduling

### Before Creating Shifts

1. Open availability matrix
2. Check who's available
3. Note time restrictions
4. Plan shifts accordingly

### Scheduling Considerations

| Scenario | Action |
|----------|--------|
| Staff unavailable | Don't assign to that day |
| Time restrictions | Ensure shift fits their hours |
| Pending time off | Wait for approval or plan backup |
| No availability set | Confirm with staff before assigning |

### Conflict Detection

The system warns when:
- Assigning to unavailable day
- Shift outside available hours
- Overlap with time off request

## Filtering the Matrix

### By Role

View availability for specific roles only.

### By Status

Filter to show:
- All staff
- Active only
- Specific employment types

### By Availability

Find staff available on specific days.

## Legend

The page includes a legend explaining:
- Available (green checkmark)
- Unavailable (red X)
- Time Off Approved (yellow minus)
- Time Off Pending (orange clock)
- Not Set (gray dash)

## Permissions

| Role | View All | Set Own | Approve Time Off |
|------|----------|---------|------------------|
| Owner | Yes | Yes | Yes |
| Admin | Yes | Yes | Yes |
| Manager | Yes | Yes | Yes |
| HR | Yes | Yes | Yes |
| Staff | Own only | Yes | No |

## Best Practices

### For Staff

- Keep availability up to date
- Submit time-off requests early
- Provide accurate time ranges
- Update when circumstances change

### For Schedulers

- Check availability before scheduling
- Respect staff preferences when possible
- Address pending requests promptly
- Document scheduling decisions

### For Managers

- Review availability weekly
- Follow up on "Not Set" entries
- Process time-off requests quickly
- Maintain fair scheduling practices

## Common Patterns

### Full-Time Staff

```
Mon-Fri: Available (09:00-17:00)
Sat-Sun: Unavailable
```

### Part-Time Staff

```
Fri-Sat: Available (18:00-23:00)
Other days: Unavailable
```

### Flexible Staff

```
All days: Available
(No time restrictions)
```

### Seasonal Preferences

```
Oct: All weekends preferred
Nov-Dec: Unavailable
Jan-Sep: Available weekends
```

## Troubleshooting

### Availability Not Saving

- Check network connection
- Verify required fields
- Refresh and retry
- Contact admin if persists

### Staff Not Showing

- Verify active employment status
- Check role assignments
- Confirm attraction assignment

### Time Off Not Appearing

- Verify request was submitted
- Check pending requests queue
- Confirm correct date range

### Matrix Not Loading

- Refresh the page
- Check for error messages
- Verify permissions
- Contact support if needed
