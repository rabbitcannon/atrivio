# Shift Templates

Create reusable shift patterns to speed up schedule creation.

![Shift Templates](/docs/screenshots/shift-templates.png)

## Overview

Shift templates allow you to define common shift patterns that can be quickly applied to your schedule, saving time when creating recurring shifts.

## Creating a Template

1. Navigate to **Schedule** → **Templates**
2. Click **+ New Template**
3. Fill in the template details:
   - **Name**: A descriptive name (e.g., "Evening Scare Actor")
   - **Start Time**: Default start time
   - **End Time**: Default end time
   - **Position**: The role for this shift
   - **Color**: Visual identifier on the calendar
4. Click **Save Template**

## Template Settings

| Setting | Description |
|---------|-------------|
| Name | Descriptive name for the template |
| Start Time | Default shift start time |
| End Time | Default shift end time |
| Duration | Calculated automatically |
| Position | Job role for this shift type |
| Color | Calendar display color |
| Notes | Default instructions |

## Using Templates

### Apply to Single Day

1. Click on a date in the calendar
2. Click **Use Template**
3. Select the template
4. Adjust details if needed
5. Save the shift

### Apply to Multiple Days

1. Select a date range
2. Choose **Apply Template**
3. Select the template
4. Choose which days of the week
5. Review and confirm

## Managing Templates

### Edit a Template

1. Go to **Schedule** → **Templates**
2. Click the template to edit
3. Update the settings
4. Save changes

Note: Changes to templates don't affect existing shifts.

### Delete a Template

1. Go to **Schedule** → **Templates**
2. Click the delete icon on the template
3. Confirm deletion

Existing shifts created from this template remain unchanged.

## Best Practices

1. **Name clearly**: Use descriptive names like "Weekend Night Shift" or "Box Office Morning"
2. **Color code**: Use consistent colors for similar shift types
3. **Include notes**: Add common instructions to templates
4. **Review regularly**: Update templates as your operations change
