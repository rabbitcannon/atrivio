# Scheduling Overview

Manage staff schedules, shifts, and availability for your attraction.

## Overview

The Scheduling module provides comprehensive tools for creating and managing staff schedules. Build shifts, track availability, handle swap requests, and use templates to streamline schedule creation.

## Accessing Scheduling

1. Navigate to **Schedule** in the sidebar
2. View the scheduling dashboard with quick stats and navigation

## Dashboard Quick Stats

| Stat | Description |
|------|-------------|
| This Week | Number of shifts scheduled this week |
| Unassigned | Shifts that need staff coverage |
| Templates | Active shift templates available |
| Hours | Total hours scheduled this week |

## Scheduling Workflow

### Weekly Schedule Creation

1. **Review Availability**: Check staff availability for the week
2. **Use Templates**: Apply templates to generate base schedule
3. **Fill Gaps**: Manually create shifts for special needs
4. **Assign Staff**: Assign available staff to shifts
5. **Publish**: Change shift status to published
6. **Notify**: Staff receive their schedule

### Managing Changes

1. **Staff Requests**: Handle swap/drop/pickup requests
2. **Review**: Evaluate request against coverage needs
3. **Approve/Reject**: Process request with notes
4. **Update Schedule**: System updates assignments automatically

## Shift Statuses

| Status | Description |
|--------|-------------|
| Draft | Initial creation, not visible to staff |
| Scheduled | Planned but not finalized |
| Published | Visible to staff, awaiting confirmation |
| Confirmed | Staff has acknowledged the shift |
| Canceled | Shift has been canceled |

## Feature Requirements

Scheduling requires the **Pro** subscription tier or higher.

## Best Practices

### Schedule Planning

- Create schedules at least 2 weeks in advance
- Review availability before scheduling
- Use templates for consistent patterns
- Keep draft shifts until finalized

### Staff Communication

- Publish schedules promptly
- Respond to swap requests within 24 hours
- Document rejection reasons
- Send reminders for upcoming shifts

### Coverage Management

- Monitor unassigned shifts daily
- Cross-train staff for multiple roles
- Maintain backup staff list
- Track patterns in coverage gaps

## Troubleshooting

### Cannot Create Shifts

- Verify Pro subscription is active
- Check role permissions
- Ensure attraction is selected
- Refresh and try again

### Staff Not Showing

- Confirm staff has active status
- Check if staff is assigned to attraction
- Verify availability is set
- Review role assignments

### Templates Not Generating

- Ensure templates are active
- Verify date range is valid
- Check role assignments
- Review attraction selection
