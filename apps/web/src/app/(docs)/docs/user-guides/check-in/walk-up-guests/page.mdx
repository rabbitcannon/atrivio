# Walk-Up Guests

Process on-site ticket purchases and check-ins.

## Overview

Walk-up guests are visitors who purchase tickets at the venue rather than online. The check-in system supports quick on-site sales and immediate entry.

## Walk-Up Process

### 1. Check Availability

Before selling:
1. View current capacity on the check-in dashboard
2. Confirm tickets are available
3. Check for any time slot restrictions

### 2. Create Walk-Up Sale

1. Click **Walk-Up** or **Quick Sale**
2. Select ticket type(s)
3. Enter quantity
4. Apply any discounts (if authorized)
5. Process payment

### 3. Complete Entry

After purchase:
1. Ticket is automatically validated
2. Guest can enter immediately
3. Capacity is updated

## Payment Options

### Card Payment

1. Enter sale amount
2. Process via card reader
3. Receipt prints/emails automatically

### Cash Payment

1. Enter sale amount
2. Record cash received
3. Provide change if needed
4. Manual receipt if required

## Walk-Up Settings

Managers can configure:

| Setting | Description |
|---------|-------------|
| Enabled | Allow walk-up sales |
| Payment Types | Cash, card, or both |
| Available Tickets | Which types can be sold |
| Discounts | Authorized discount options |

## Group Walk-Ups

For larger groups:

1. Click **Group Sale**
2. Enter group size
3. Apply group discount if available
4. Process payment
5. Print group ticket or multiple individual tickets

## Handling Special Cases

### Sold Out Events

If capacity is reached:
1. Inform guest politely
2. Offer alternative time slots
3. Suggest online booking for future

### Cash Drawer Management

- Start shift with opening count
- Record all cash transactions
- End shift with closing count
- Report discrepancies

### Comp Tickets

For complimentary entries:

1. Select **Comp Ticket**
2. Enter reason
3. Get manager approval (if required)
4. Record entry

## Reporting

Walk-up data includes:
- Daily walk-up revenue
- Walk-up vs. online ratio
- Popular ticket types
- Peak walk-up times

## Best Practices

1. **Stay updated on capacity**: Check dashboard frequently
2. **Know your ticket types**: Be ready to explain options
3. **Process payments carefully**: Verify amounts
4. **Keep line moving**: Be efficient during busy times
5. **Handle cash securely**: Follow cash handling procedures

## Troubleshooting

### Payment Failed

1. Verify card details
2. Try again
3. Offer alternative payment method

### System Offline

1. Use backup paper tickets
2. Record sales manually
3. Enter into system when online
