# Staff Profiles

View and manage detailed information for each team member.

## Overview

Staff profiles contain comprehensive information about each team member, including employment details, personal information, emergency contacts, and quick access to skills, certifications, and time tracking.

## Accessing a Profile

1. Navigate to **Staff** in the sidebar
2. Click on a staff member's name or **View Profile**
3. The profile page opens with multiple tabs

## Profile Tabs

| Tab | Description |
|-----|-------------|
| Overview | Quick summary and key information |
| Skills | Proficiency ratings and abilities |
| Certifications | Licenses and certifications |
| Time Tracking | Individual time history |

## Overview Tab

The Overview tab displays at-a-glance information:

### Header Section

- Staff member's name and avatar
- Current role badge
- Employment status indicator
- Quick action buttons

### Employment Information

| Field | Description |
|-------|-------------|
| Employee ID | Unique identifier (e.g., EMP-001) |
| Status | Active, Inactive, On Leave, Terminated |
| Employment Type | Full-time, Part-time, Seasonal, Contractor |
| Hire Date | When staff member started |
| Hourly Rate | Pay rate (visible to authorized roles) |

### Personal Information

| Field | Description |
|-------|-------------|
| Email | Contact email address |
| Phone | Contact phone number |
| Shirt Size | For uniforms and merchandise |

### Emergency Contact

| Field | Description |
|-------|-------------|
| Name | Emergency contact person |
| Phone | Emergency contact number |
| Relationship | Relation to staff member |

### Notes

Free-form notes field for additional information about the staff member.

## Editing a Profile

1. Click **Edit** on the profile page
2. Modify the desired fields
3. Click **Save Changes**

### Editable Fields

**Employment Info:**
- Employee ID
- Status
- Employment Type
- Hourly Rate

**Personal Info:**
- Shirt Size (XS, S, M, L, XL, 2XL, 3XL)

**Emergency Contact:**
- Contact Name
- Contact Phone
- Relationship

**Notes:**
- Additional notes about the staff member

### Non-Editable Fields

Some fields cannot be changed from the profile:
- Email (managed through account settings)
- Name (managed through user profile)
- Role (managed through staff assignment)

## Skills Preview

The Overview tab shows a quick view of the staff member's skills with proficiency badges. Click "View All" or the Skills tab for complete details.

## Certifications Preview

Active certifications are shown with expiration status:

| Status | Indicator | Meaning |
|--------|-----------|---------|
| Valid | Green | Current and valid |
| Expiring Soon | Yellow | Expires within 30 days |
| Expired | Red | Past expiration date |

## Profile Permissions

| Role | View Profile | Edit Profile | View Hourly Rate |
|------|--------------|--------------|------------------|
| Owner | Yes | Yes | Yes |
| Admin | Yes | Yes | Yes |
| Manager | Yes | Limited | No |
| HR | Yes | Yes | Yes |
| Others | Own only | No | No |

## Best Practices

### Profile Completeness

- Fill in all available fields for complete records
- Keep emergency contacts up to date
- Verify information during onboarding

### Status Management

- Update status promptly when circumstances change
- Document status change reasons in notes
- Review statuses before each season

### Privacy Considerations

- Only authorized roles can view sensitive info
- Hourly rate is hidden from most roles
- Emergency contacts are for emergencies only

## Troubleshooting

### Cannot Edit Profile

- Verify you have the required role permissions
- Check if the staff member is in an editable status
- Contact admin if permissions seem incorrect

### Missing Information

- Some fields may not be set yet
- Contact HR to update missing required info
- Check if field is visible to your role

### Profile Not Loading

- Refresh the page
- Check your network connection
- Verify staff member still exists in system
