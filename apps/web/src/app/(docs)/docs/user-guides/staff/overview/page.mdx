# Staff Directory

Manage your team members, invitations, and staff roster.

## Overview

The Staff Directory is your central hub for managing all team members across your attraction. View active staff, invite new team members, and access individual profiles for detailed management.

## Accessing Staff Directory

1. Navigate to **Staff** in the sidebar
2. View the full roster of team members

## Staff List

The staff directory displays all team members with key information:

| Column | Description |
|--------|-------------|
| Name | Staff member's full name |
| Role | Assigned role in your organization |
| Status | Current employment status |
| Employment Type | Full-time, part-time, etc. |
| Actions | Quick access to profile and edit |

## Staff Statuses

| Status | Badge | Description |
|--------|-------|-------------|
| Active | Green | Currently employed and working |
| Inactive | Gray | Temporarily not working |
| On Leave | Yellow | Approved leave of absence |
| Terminated | Red | No longer employed |

## Adding New Staff

Staff members are added through an invitation system, ensuring proper onboarding:

1. Click **Add Staff** from the directory
2. Enter the staff member's email address
3. Select their initial role
4. Click **Send Invitation**

The invited person will receive an email with instructions to create their account and join your organization.

### Available Roles

| Role | Description |
|------|-------------|
| Manager | Full operational access, can manage staff |
| HR | Human resources, can manage profiles and certifications |
| Box Office | Ticketing and sales access |
| Finance | Financial reports and payment access |
| Actor | Performance staff with basic access |
| Scanner | Check-in and scanning access only |

## Filtering Staff

Use the filter options to find specific team members:

- **By Role**: Filter by manager, actor, scanner, etc.
- **By Status**: View only active, inactive, or on leave staff
- **By Employment Type**: Full-time, part-time, seasonal, contractor

## Searching Staff

Use the search box to quickly find staff by:
- First or last name
- Email address
- Employee ID

## Staff Actions

From the directory, you can:

| Action | Description |
|--------|-------------|
| View Profile | See complete staff details |
| Edit | Modify staff information |
| Quick Actions | Access common operations |

## Role Permissions

Different roles have different access levels:

| Role | Staff Directory | Invite Staff | Edit Profiles |
|------|-----------------|--------------|---------------|
| Owner | Full access | Yes | Yes |
| Admin | Full access | Yes | Yes |
| Manager | View only | Yes | Limited |
| HR | Full access | Yes | Yes |
| Others | View only | No | No |

## Pending Invitations

View and manage outstanding invitations:

1. Click **Pending Invitations** tab
2. See all unaccepted invitations
3. Resend or cancel as needed

## Best Practices

### Invitation Management

- Send invitations during business hours for faster acceptance
- Follow up on pending invitations after 48 hours
- Cancel invitations if no longer needed

### Role Assignment

- Assign the minimum role needed for job duties
- Review roles periodically as responsibilities change
- Document role changes for audit purposes

### Status Updates

- Update status promptly when staff circumstances change
- Use "On Leave" for temporary absences
- Only use "Terminated" for permanent departures

## Troubleshooting

### Invitation Not Received

- Check spam/junk folder
- Verify email address spelling
- Resend invitation if needed
- Contact support if problem persists

### Duplicate Staff Entry

- Search before inviting to avoid duplicates
- If duplicate exists, contact admin to merge records
- Inactive duplicates can be terminated

### Permission Issues

- Verify your role has required permissions
- Contact organization admin for access upgrades
- Check if staff member is in correct status
