# Staff Skills

Track and manage staff member abilities and proficiency levels.

## Overview

Skills tracking allows you to document what each staff member can do and how proficient they are. This helps with scheduling, training assignments, and ensuring the right people are in the right positions.

## Accessing Skills

1. Navigate to **Staff** in the sidebar
2. Click on a staff member's name
3. Select the **Skills** tab

## Skill List

Each skill entry shows:

| Component | Description |
|-----------|-------------|
| Skill Name | The ability or competency |
| Proficiency | Star rating (1-5) |
| Actions | Edit or remove skill |

## Proficiency Levels

Skills are rated on a 5-star scale:

| Stars | Level | Description |
|-------|-------|-------------|
| ★☆☆☆☆ | Beginner | Just learning, needs supervision |
| ★★☆☆☆ | Basic | Can perform with guidance |
| ★★★☆☆ | Intermediate | Competent, works independently |
| ★★★★☆ | Advanced | Highly skilled, can train others |
| ★★★★★ | Expert | Master level, go-to resource |

## Adding a Skill

1. Click **Add Skill** on the Skills tab
2. Select a skill from the suggestions or enter a custom skill
3. Set the proficiency level (1-5 stars)
4. Click **Add Skill**

## Editing a Skill

1. Find the skill in the list
2. Click **Edit**
3. Adjust the proficiency level
4. Click **Save**

## Removing a Skill

1. Find the skill in the list
2. Click **Remove** or the trash icon
3. Confirm the removal

Note: Removing a skill doesn't delete it from the system, only from this staff member's profile.

## Skill Assessment

### Initial Assessment

When adding a new staff member:
1. Review their application or resume
2. Conduct practical assessment if needed
3. Add skills with appropriate starting levels
4. Plan training for skill improvement

### Ongoing Updates

Skills should be updated when:
- Staff completes training
- Performance review indicates improvement
- New abilities are demonstrated
- Certification is earned

## Skill Development

### Training Paths

Create skill development paths for staff:

1. Identify target skill level
2. Assign training or mentorship
3. Set timeline for improvement
4. Reassess after training
5. Update proficiency level

### Mentorship Pairing

Pair staff for skill development:
- Expert (5 stars) mentors Beginners (1-2 stars)
- Advanced (4 stars) works with Intermediate (3 stars)
- Document progress and update skills

## Best Practices

### Accurate Assessment

- Rate skills honestly based on actual ability
- Use practical tests when possible
- Get input from supervisors who work with staff
- Avoid inflating or deflating ratings

### Regular Updates

- Review skills during performance reviews
- Update after training completion
- Adjust based on observed performance
- Remove skills no longer relevant

### Consistent Standards

- Define what each level means for your team
- Apply same standards across all staff
- Document assessment criteria
- Train managers on consistent evaluation

## Troubleshooting

### Cannot Add Skills

- Verify you have manager or higher permissions
- Check if staff member is in active status
- Refresh the page and try again

### Skill Not Saving

- Ensure skill name is not empty
- Verify proficiency is selected (1-5)
- Check network connection
- Contact support if issue persists

### Duplicate Skills

- Search existing skills before adding
- Edit existing skill instead of adding duplicate
- Remove duplicates if found
