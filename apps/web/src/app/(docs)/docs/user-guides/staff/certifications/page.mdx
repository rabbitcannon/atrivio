# Staff Certifications

Manage licenses, certifications, and compliance tracking for staff members.

## Overview

Certifications track important licenses and credentials that staff members hold. The system monitors expiration dates, manages verification, and helps ensure your team stays compliant with required certifications.

## Accessing Certifications

1. Navigate to **Staff** in the sidebar
2. Click on a staff member's name
3. Select the **Certifications** tab

## Certification List

Each certification displays:

| Component | Description |
|-----------|-------------|
| Type | Category of certification |
| Certificate Number | Official ID or number |
| Issued Date | When certification was earned |
| Expiration Date | When it expires (if applicable) |
| Status | Current validity status |
| Verification | Verification status and details |
| Actions | Edit, verify, or remove |

## Certification Status

| Status | Badge | Description |
|--------|-------|-------------|
| Valid | Green | Current and active |
| Expiring Soon | Yellow | Expires within 30 days |
| Expired | Red | Past expiration date |
| No Expiry | Blue | Does not expire |

## Adding a Certification

1. Click **Add Certification** on the Certifications tab
2. Select the certification type
3. Enter the certificate number (if applicable)
4. Set the issued date
5. Set the expiration date (leave blank if no expiry)
6. Click **Add Certification**

### Required Fields

| Field | Required | Notes |
|-------|----------|-------|
| Type | Yes | Select from list |
| Certificate Number | No | Official ID number |
| Issued Date | Yes | When earned |
| Expiration Date | No | Leave blank if none |

## Editing a Certification

1. Find the certification in the list
2. Click **Edit**
3. Modify the fields as needed
4. Click **Save Changes**

## Removing a Certification

1. Find the certification in the list
2. Click **Remove** or the trash icon
3. Confirm the removal

**Warning:** Removing a certification deletes the record. Consider keeping expired certifications for historical records.

## Verification Workflow

Certifications can be verified by authorized staff:

### Verification Process

1. Staff member adds certification
2. Manager or HR reviews documentation
3. Click **Verify** on the certification
4. Certification marked as verified with:
   - Verified by (who verified)
   - Verified at (when verified)

### Verification Status

| Status | Description |
|--------|-------------|
| Unverified | Not yet reviewed |
| Verified | Confirmed by authorized staff |

### Who Can Verify

| Role | Can Verify |
|------|------------|
| Owner | Yes |
| Admin | Yes |
| Manager | Yes |
| HR | Yes |
| Others | No |

## Expiration Tracking

The system automatically tracks certification expirations:

### Expiration Alerts

| Timeline | Alert |
|----------|-------|
| 30+ days | Status: Valid (green) |
| 1-30 days | Status: Expiring Soon (yellow) |
| 0 days | Status: Expired (red) |

### Renewal Reminders

Set up reminders for upcoming expirations:
- Dashboard shows expiring certifications
- Notifications can be sent to staff
- Managers see team expiration summary

## Best Practices

### Documentation

- Keep copies of certification documents
- Record certificate numbers accurately
- Document verification with dates

### Tracking

- Review expirations monthly
- Set renewal reminders 60 days out
- Plan training before busy seasons

### Verification

- Verify all certifications within 7 days
- Request documentation for verification
- Update verification if recertified

### Compliance

- Define required certifications by position
- Don't schedule uncertified staff in certified positions
- Track compliance metrics regularly

## Troubleshooting

### Certification Not Showing

- Verify it was saved successfully
- Refresh the page
- Check if filtered out by status

### Expiration Date Issues

- Ensure date format is correct
- Check if certification has no expiry
- Verify issued date is before expiration

### Cannot Verify

- Check your role permissions
- Ensure certification exists
- Contact admin if permissions seem wrong

### Duplicate Certifications

- Remove old expired versions
- Keep most recent certification
- Document renewal in notes
