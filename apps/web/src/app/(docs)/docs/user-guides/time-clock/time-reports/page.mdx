# Time Reports

View and export time tracking data for payroll and analysis.

## Overview

Time reports provide detailed insights into staff hours worked, helping managers with payroll processing and scheduling optimization.

## Viewing Time Reports

### Staff View

Individual staff members can view:
- Daily time entries
- Weekly hour summaries
- Pay period totals

### Manager View

Managers and admins can access:
- Team-wide time reports
- Individual staff histories
- Export options for payroll

## Report Types

### Daily Report

Shows all time entries for a specific date:
- Clock in/out times
- Break durations
- Total hours worked

### Weekly Summary

Aggregated view of the week:
- Daily totals
- Weekly total hours
- Overtime calculations (if applicable)

### Pay Period Report

Comprehensive report for payroll:
- All entries within the pay period
- Total regular hours
- Overtime hours (if configured)
- Export to CSV/PDF

## Filtering & Sorting

Filter reports by:
- **Date range**: Custom date selection
- **Staff member**: Individual or team view
- **Attraction**: Filter by work location
- **Status**: Approved, pending, or flagged entries

## Exporting Data

Export time data for external payroll systems:

1. Select your date range
2. Choose the staff members to include
3. Click **Export**
4. Select format (CSV or PDF)

## Time Entry Corrections

### Requesting Corrections

Staff can request corrections for:
- Missed clock in/out
- Incorrect times
- Break adjustments

### Approving Corrections

Managers can:
- Review correction requests
- Approve or deny changes
- Add notes for auditing
