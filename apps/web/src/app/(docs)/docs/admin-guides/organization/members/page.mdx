# Members

Manage team members and their access levels within your organization.

## Overview

The members page allows you to invite new team members, assign roles, and manage access permissions across your organization.

## Viewing Members

The member list shows:
- Name and email
- Role
- Status (active, pending, suspended)
- Last active date

## Inviting Members

### Send Invitation

1. Click **Invite Member**
2. Enter email address
3. Select role
4. Add personal message (optional)
5. Click **Send Invite**

### Invitation Status

| Status | Description |
|--------|-------------|
| Pending | Invite sent, awaiting response |
| Accepted | User joined organization |
| Expired | Invite not used within 7 days |

### Resend Invitation

1. Find pending invite
2. Click **Resend**
3. New email sent, timer resets

## Role Assignment

### Available Roles

| Role | Access Level |
|------|--------------|
| Owner | Full access, cannot be removed |
| Admin | Full access, manage members |
| Manager | Operational access |
| HR | Staff management only |
| Finance | Financial data only |
| Box Office | Ticketing and check-in |
| Actor | Time clock and schedule view |
| Scanner | Check-in only |

### Change Role

1. Click on the member
2. Select new role
3. Confirm the change

### Role Permissions

Each role has specific permissions:

**Admin**
- ✅ All features
- ✅ Member management
- ✅ Settings access

**Manager**
- ✅ Staff management
- ✅ Scheduling
- ✅ Ticketing
- ❌ Billing access

**Actor/Staff**
- ✅ Time clock
- ✅ View schedule
- ❌ Administrative features

## Managing Members

### View Member Details

Click a member to see:
- Profile information
- Role history
- Activity log
- Assigned attractions

### Suspend Member

Temporarily disable access:
1. Open member profile
2. Click **Suspend**
3. Select reason
4. Confirm

Member can't access until reactivated.

### Remove Member

Permanently remove from organization:
1. Open member profile
2. Click **Remove**
3. Confirm removal

Data associated with member is retained.

## Bulk Actions

### Bulk Invite

1. Click **Bulk Invite**
2. Upload CSV with emails and roles
3. Review the list
4. Send all invites

### Export Members

1. Click **Export**
2. Choose format
3. Download member list

## Security

### Two-Factor Authentication

Encourage or require 2FA:
- View 2FA status for each member
- Send reminders to enable
- Require for admin roles

### Session Management

View active sessions:
- Devices logged in
- Last activity
- Force logout if needed

## Audit Log

Track member changes:
- Role changes
- Invitations sent
- Members removed
- Access changes
