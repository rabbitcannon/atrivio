# Branding

Customize your organization's visual identity across the platform.

## Overview

Branding settings allow you to personalize the look and feel of your dashboard, storefront, and customer-facing pages.

## Logo Settings

### Primary Logo

Your main logo appears in:
- Dashboard header
- Emails to customers
- Printed tickets
- Storefront header

**Requirements:**
- Format: PNG, JPG, or SVG
- Recommended size: 200x60 pixels minimum
- Transparent background recommended

### Favicon

The small icon in browser tabs:
- Format: ICO, PNG, or SVG
- Size: 32x32 or 64x64 pixels

## Color Scheme

### Primary Colors

| Color | Usage |
|-------|-------|
| Primary | Buttons, links, accents |
| Secondary | Secondary actions |

### Brand Colors

Apply your brand colors to:

## Best Practices

### Consistency

1. Use same colors everywhere
2. Maintain logo sizing
3. Keep fonts consistent
4. Match your website

### File Optimization

1. Compress images
2. Use appropriate formats
3. Test on mobile devices
4. Verify email rendering

## Preview & Testing

Before publishing changes:

1. Preview in dashboard
2. Send test email
3. View test ticket
4. Check storefront
