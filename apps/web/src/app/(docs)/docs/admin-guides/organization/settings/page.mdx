# Organization Settings

Configure your organization's core settings and preferences.

![Organization Settings](/docs/screenshots/org-settings.png)

## Overview

Organization settings control the fundamental configuration of your attraction, including business information, operational settings, and feature preferences.

## General Settings

### Business Information

| Setting | Description |
|---------|-------------|
| Organization Name | Your business name |
| Slug | URL-friendly identifier |
| Description | Brief description of your attraction |
| Logo | Organization logo (displayed in dashboard) |

### Contact Information

| Setting | Description |
|---------|-------------|
| Email | Primary contact email |
| Phone | Business phone number |
| Website | Public website URL |
| Address | Physical location |

## Operational Settings

### Timezone

Set your local timezone for:
- Scheduling
- Time clock
- Event times
- Reports

### Business Hours

Define your operating hours:
1. Set hours for each day of the week
2. Mark days as closed if applicable
3. Set seasonal variations

### Currency

Select your currency for:
- Ticket pricing
- Payroll
- Financial reports

## Feature Settings

### Enabled Features

Toggle features based on your plan:
- ✅ Ticketing
- ✅ Check-In
- ✅ Time Tracking
- ⬜ Scheduling (Pro)
- ⬜ Inventory (Pro)

### Notifications

Configure notification preferences:
- Email notifications
- SMS alerts (Enterprise)
- Push notifications

## Attractions

Manage your attractions/venues:

### Add Attraction

1. Click **+ Add Attraction**
2. Enter attraction details
3. Configure capacity settings
4. Save

### Edit Attraction

1. Select the attraction
2. Update settings
3. Save changes

## Danger Zone

Critical settings requiring caution:

### Transfer Ownership

Transfer organization to another admin:
1. Enter new owner's email
2. Confirm the transfer
3. New owner accepts

### Delete Organization

Permanently delete organization:
1. Enter organization name to confirm
2. Understand all data will be deleted
3. This action cannot be undone

## Audit Log

View changes to organization settings:
- Who made the change
- What was changed
- When it occurred
