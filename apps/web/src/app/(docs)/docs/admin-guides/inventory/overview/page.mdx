# Inventory Overview

Track costumes, props, equipment, and supplies across your attractions.

## Overview

The Inventory Management system helps you track and manage all physical assets used in your attractions. Monitor stock levels, check out items to staff, and maintain a complete transaction history.

**Note:** Inventory Management is a Pro-tier feature.

## Accessing Inventory

Navigate to **Inventory** in the sidebar to access the inventory dashboard.

## Dashboard

The inventory dashboard provides a quick overview of your inventory status.

### Quick Stats

| Metric | Description |
|--------|-------------|
| Total Items | Number of unique items tracked |
| Checked Out | Active checkouts to staff members |
| Low Stock | Items at or below minimum quantity |
| Overdue | Checkouts past their due date |

### Low Stock Alerts

When items fall to or below their minimum quantity, they appear in the Low Stock Alerts section. Each alert shows:

- Item name and SKU
- Current quantity vs. minimum quantity

Click an alert to view all low stock items.

## Item Types

Inventory items are categorized by type:

| Type | Description | Checkout Required |
|------|-------------|-------------------|
| **Costume** | Clothing, masks, wigs | Yes |
| **Prop** | Hand props, set pieces | Yes |
| **Equipment** | Tools, electronics, machinery | Yes |
| **Supply** | Consumables, makeup, batteries | No |

Items with "Checkout Required" must be checked out to staff before use.

## Common Workflows

### Adding New Inventory

1. Go to **Inventory** → **Items**
2. Click **Add Item**
3. Enter SKU, name, and type
4. Set initial quantity and minimum
5. Optionally assign to category/attraction
6. Save

### Restocking Items

1. Go to **Inventory** → **Items**
2. Find the item
3. Click **Adjust Quantity**
4. Select "Purchase" as the type
5. Enter quantity added
6. Add notes (e.g., order number)
7. Save

### End-of-Season Inventory

1. Review low stock alerts
2. Process overdue checkouts
3. Adjust quantities for damaged/lost items
4. Update categories as needed

## Best Practices

### Item Organization

- Use consistent SKU formats (e.g., MASK-001, PROP-042)
- Create logical category hierarchies
- Assign items to specific attractions when possible
- Set appropriate minimum quantities

### Checkout Management

- Set realistic due dates
- Check item condition at checkout and return
- Follow up on overdue items promptly
- Use notes to document any issues

### Stock Monitoring

- Review low stock alerts regularly
- Order supplies before they run out
- Track seasonal inventory needs
- Document damaged or lost items
