# Storefront Pages

Create and manage custom pages for your storefront.

## Overview

Add custom pages to your storefront for additional content like event information, rules, directions, or special promotions.

## Creating Pages

### Add a New Page

1. Go to **Storefront** → **Pages**
2. Click **+ New Page**
3. Enter page details:
   - **Title**: Page heading
   - **Slug**: URL path (e.g., "rules" creates /rules)
   - **Content**: Page content
4. Click **Save**

### Page Editor

The editor supports:
- **Headings**: H1, H2, H3
- **Text formatting**: Bold, italic, lists
- **Links**: Internal and external
- **Images**: Upload or embed
- **Videos**: YouTube, Vimeo embeds
- **Custom HTML**: For advanced users

## Page Settings

### Basic Settings

| Setting | Description |
|---------|-------------|
| Title | Page title and heading |
| Slug | URL path |
| Meta Description | SEO description |
| Featured Image | Social sharing image |

### Visibility

Control page visibility:
- **Published**: Visible to everyone
- **Draft**: Hidden from public
- **Password Protected**: Requires password

### Navigation

Add page to navigation:
- ✅ Show in header menu
- ✅ Show in footer
- Set menu order

## Page Templates

### Available Templates

Start with pre-built layouts:
- **Blank**: Start from scratch
- **Content**: Text with sidebar
- **Landing**: Full-width with sections
- **Gallery**: Image-focused layout

### Customizing Templates

1. Select a template
2. Modify content
3. Adjust layout
4. Save changes

## Managing Pages

### Edit Pages

1. Go to **Storefront** → **Pages**
2. Click the page to edit
3. Make changes
4. Save or Publish

### Reorder Pages

Drag and drop to reorder:
- Menu order
- Footer order

### Delete Pages

1. Open the page
2. Click **Delete**
3. Confirm deletion

Deleted pages can be restored within 30 days.

## SEO for Pages

Optimize each page:

### Meta Tags

- **Title Tag**: Shows in browser/search
- **Description**: Search result preview
- **Keywords**: Relevant terms

### URL Structure

Keep URLs clean:
- ✅ /rules
- ✅ /how-to-find-us
- ❌ /page?id=123
